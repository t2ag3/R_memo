Demonstration for usage

Plot without mytheme::theme_paper()

iris %>% ggplot(aes(x = Sepal.Length, y = Sepal.Width, color = Species)) +
  geom_point()

Plot with mytheme::theme_paper()

iris %>% ggplot(aes(x = Sepal.Length, y = Sepal.Width, color = Species)) +
  geom_point() +
  theme_paper()
